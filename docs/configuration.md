# Configuration

## Configuration file location

The registry location and the list of watched ROMs folders are stored in a configuration file, whose location can be customized.

| Environment variable | Purpose | Default value | Required |
|---|---|---|---|
| `BATOCERA_SCRAP_MANAGER_CONFIG` | Path to the configuration file to use | The OS user config directory, under `batocera-scrap-manager/config.json` | No |

This variable is mainly useful for isolating a test environment's configuration or running several configurations side by side on the same machine.

## Configuration content

The configuration file itself is not meant to be edited by hand: it is managed via the `batocera-scrap-manager config set-registry` and `batocera-scrap-manager config add-roms-folder` subcommands (see the README). It contains:

- the location of the registry where already-scraped game metadata is centralized;
- the list of Batocera ROMs folders to watch.

If the configuration file does not exist yet, the tool behaves as if no registry and no ROMs folder were configured, without error.
